ANUCDE Chatbot Migration Guide
Follow these steps to move the project from this computer to another system using a USB drive.

1. Prepare the Project for Compression
Before zipping the folder, ensure you have a clean version.

IMPORTANT

Do NOT include the venv folder in your zip file. It contains thousands of files that are specific to your current system and will not work on a different computer. It will also make the zip file much larger and slower to copy.

Steps to Compress:
Navigate to e:\anucde-cms\.
Right-click the anucde_chatbot folder.
Select Compress to ZIP file (or use WinRAR/7-Zip).
If you want to be extra clean, manually exclude the venv folder before zipping.

2. Setup on the New System
On the target computer, you need to install the following software:

Required Software:
Python (3.10 or higher): Download from python.org. During installation, make sure to check the box "Add Python to PATH".
Web Browser: Any modern browser (Chrome, Edge, Firefox) to view the chatbot interface.

3. Extract and Run
Copy the ZIP file from your USB drive to the new system.
Extract the ZIP file to a location of your choice (e.g., C:\anucde_chatbot).
Open a Terminal or Command Prompt in that folder.
Set up the Backend:
powershell
# Navigate to the backend folder
cd backend
# Create a new virtual environment
python -m venv venv
# Activate the virtual environment
.\venv\Scripts\activate
# Install the required libraries
pip install -r requirements.txt
Run the Backend:
powershell
# While still in the backend folder and with venv activated
python main.py
The server should start on http://0.0.0.0:8000.

4. Open the Chatbot
Navigate to the frontend folder in your file explorer.
Double-click 
chatbot.html
 to open it in your browser.
The chatbot should now be connected to your local backend and ready to use!
Troubleshooting
Connection Error: Ensure the backend terminal is still running.
Missing Module: If you get a "ModuleNotFoundError", ensure you ran pip install -r requirements.txt while the venv was active.
